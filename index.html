I hope to pursue graduate research on topics that combine clear problem formulation, mechanism design, and rigorous experimentation.

大模型安全遗忘课题研究

Safety unlearning for multimodal large language models

2026.01 – 2026.03
第二作者 / Second Author ACM MM 2026 已投稿 / Submitted CCF A

围绕多模态大模型中的安全遗忘问题展开研究，重点关注如何在抑制有害行为的同时减少对良性样本的过度拒答。我主要参与实验设计、云端训练、图表绘制与数据分析。

This project studies selective safety unlearning for multimodal models, with emphasis on reducing harmful behavior while preserving benign utility. I contributed to experiment design, cloud-based training, figure production, and empirical analysis.

大模型多轮拒答校准课题研究

Refusal calibration under benign multi-turn dialogue history

2026.01 – 2026.03
第一作者 / First Author ICIC 2026 已投稿 / Submitted CCF C

该项目研究良性多轮对话历史对模型拒答行为的影响，关注安全请求在特定上下文下被错误拒答的问题。我完成了 Python 实验实现、方法设计迭代、绘图、数据分析与全文撰写。

This project investigates history-induced refusal drift in aligned language models. I built the Python pipeline, refined the methodology, and completed the full analysis and writing as the first author.

大模型安全红队测试课题研究

Budget-efficient red teaming for LLM safety evaluation

2025.11 – 2026.01
第三作者 / Third Author IJCAI 2026 已投稿 / Submitted CCF A（投稿时） / CCF A at submission

该项目将红队测试看作预算受限下的失败发现问题，重点提升在有限查询预算中尽早发现多样化违规样例的能力。我使用 Python 结合 API 对四个大模型完成实验，并参与数据分析与图表制作。

This work reframes red teaming as budget-constrained failure discovery. I participated in experiments across four large models using Python and API-based evaluation, and contributed to analysis and visualization.

遥感视觉语言预训练课题研究

Remote sensing vision-language pre-training with hierarchical semantics

2025.12 – 2026.03
第四作者 / Fourth Author ACM MM 2026 已投稿 / Submitted CCF A

该项目关注遥感场景下视觉语言预训练中的层次语义建模问题，探索从全局场景到局部细粒度区域的统一建模方式。我主要参与实验迭代、图表设计与论文撰写。

This project explores hierarchical semantic learning for remote sensing vision-language pre-training. I contributed to experiment iteration, figure design, and project writing.
