# Academic Homepage for Pengrui Xiang

This folder contains a ready-to-publish GitHub Pages homepage.

## Files
- `index.html` — homepage content
- `style.css` — page styling
- `assets/avatar-placeholder.svg` — replace with your own headshot if desired
- `assets/Xiang_Pengrui_CV.docx` — current CV download file

## Publish on GitHub Pages
1. Create a public repository named `cuberpeng.github.io`
2. Upload all files in this folder to the repository root
3. In GitHub, open **Settings → Pages**
4. Set **Source** to **Deploy from a branch**
5. Choose **main** branch and **/root** folder
6. Wait a few minutes, then open `https://cuberpeng.github.io`

## Recommended quick edits before publishing
- Replace `assets/avatar-placeholder.svg` with your own photo and keep the same file name, or update the file path in `index.html`
- If you have a PDF CV, replace `assets/Xiang_Pengrui_CV.docx` with a PDF and update the download link text
- Update the opening sentence in the hero section once you finalize your main target research direction
- Add Google Scholar or personal email aliases later if needed

## Design choice notes
- Research summaries are written in English and avoid using unpublished paper titles
- The page is single-file and lightweight, so it is easy to edit by hand
- The content order is optimized for faculty outreach: profile → research → experience → awards → skills
